operacoes_banco.somar_saldos_em_lote(**kwargs) → int[código-fonte]

Soma valores aos saldos de múltiplas contas de uma vez. No kwargs, as chaves são os números das contas e os valores serão os montantes a serem adicionados. Se forem inseridas contas inexistentes ou valores negativos, eles serão ignorados.

Retorna:

Quantidade de contas que tiveram saldo atualizado.

Tipo de retorno:

int

operacoes_banco.somar_saldos_gerais() → float[código-fonte]

Tipo de retorno:

operacoes_banco.subtrair_saldos_em_lote(**kwargs) → int[código-fonte]

Subtrai valores aos saldos de múltiplas contas de uma vez. No kwargs, as chaves são os números das contas e os valores serão os montantes a serem adicionados. Se uma conta tiver saldo menor do que o retirado, o saque falha para ela, mas continua para as outras. Contas inexistentes, valores negativos ou tentativas de saque maiores que o saldo serão ignoradas.

Retorna:

Quantidade de contas que tiveram saldo alterado com sucesso.

Tipo de retorno:

int

operacoes_banco.trata_float(valor: str | float | int) → str[código-fonte]

Trata as entradas numéricas, garantindo que os valores sigam o padrão do banco: apenas separador decimal, sendo ele um ponto final. Se a entrada for vazia, retorna 0.

Parâmetros:

string (str | float | int) – Número a ser modificado, passado em forma de string, int ou float.

Retorna:

Valor inserido com ponto no separamento decimal e arredondamento para duas casas.

Tipo de retorno:

str
